# Warehouse Inventory Management System

## 📌 Project Description
This project is a console-based Warehouse Inventory Management System developed using Java.  
It uses the Java Collections Framework (HashMap) to manage inventory efficiently.

## 🚀 Features
- Add new items
- Remove items
- Update item quantity
- Search items by ID (case-insensitive)
- Search items by name (case-insensitive & partial search)
- Display all items
- Handles invalid input

## 🛠 Technologies Used
- Java
- HashMap (Java Collections Framework)
